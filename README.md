# Real-Time Hand Gesture Recognition System

## Overview

A real-time hand gesture recognition system built with Python using computer vision and hand landmark detection. It processes live webcam input, compares gestures with a reference database, and performs similarity-based classification for interactive gesture control.

## Features

* Real-time webcam gesture detection
* Hand landmark tracking
* Gesture similarity matching
* Reference gesture database
* Live visual feedback

## Technologies Used

* Python
* OpenCV
* MediaPipe
* NumPy

## How to Run

Install dependencies:

pip install opencv-python mediapipe numpy

Run the program:

python main.py
